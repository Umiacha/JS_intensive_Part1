# Создание простого сервера на Node.js, который отвечает на HTTP-запросы (получение / создание постов).

Cамостоятельно
- на основе jsonplaceholder реализуем CRUD для постов
- получаем все посты
- получаем конкретный пост
- создаем пост
- редактируем пост
- удаляем пост
- получаем посты пользователя
